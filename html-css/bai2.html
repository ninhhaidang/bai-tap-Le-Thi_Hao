Lê Thị Hảo - BÀI 2: DANH SÁCH VÀ LIÊN KẾT

Danh sách có thứ tự (Ordered List)

Các bước học HTML cơ bản:

  1. Học cấu trúc HTML cơ bản
  2. Tìm hiểu về các thẻ HTML
  3. Thực hành tạo trang web đơn giản
  4. Học CSS để trang trí trang web
  5. Kết hợp HTML và CSS để tạo website hoàn chỉnh

Danh sách không thứ tự (Unordered List)

Các thẻ HTML cơ bản cần biết:

  • Thẻ tiêu đề: h1, h2, h3, h4, h5, h6
  • Thẻ đoạn văn: p
  • Thẻ hình ảnh: img
  • Thẻ liên kết: a
  • Thẻ danh sách: ul, ol, li
  • Thẻ div và span
